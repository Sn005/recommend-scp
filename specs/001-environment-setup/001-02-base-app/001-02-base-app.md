## ユーザーストーリー

**ペルソナ**: 開発者
**目的**: 開発の基盤となるアプリケーション構造を作成する
**価値**: 迅速に機能開発を開始できる
**理由**: ボイラープレートを整備して開発効率を向上させたい

> 開発者として、開発の基盤となるアプリケーション構造を作成して、迅速に機能開発を開始したい。なぜならボイラープレートを整備して開発効率を向上させたいから。

## Acceptance Criteria

- [ ] 基本的なディレクトリ構造が作成されていること

- [ ] ビルドが成功すること
- [ ] 型チェックが通過すること
